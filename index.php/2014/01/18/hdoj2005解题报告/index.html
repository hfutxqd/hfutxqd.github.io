    hdoj2005解题报告

    @imxqd · Jan 18, 2014 · 1 min read

    Problem Description
    给定一个日期，输出这个日期是该年的第几天。
     


    Input
    输入数据有多组，每组占一行，数据格式为YYYY/MM/DD组成，具体参见sample input ,另外，可以向你确保所有的输入数据是合法的。
     


    Output
    对于每组输入数据，输出一行，表示该日期是该年的第几天。
     
    首先要知道该年是不是闰年,然后再计算
    代码: 
    #include  int main() { int y, m, d; while (scanf("%d/%d/%d", &y, &m, &d) != EOF) { int day[12] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31}; if ((y % 4 == 0 && y % 100 != 0) || y % 400 == 0) day[1] = 29; int sum = 0; for (int i = 0; i < m - 1; i++) sum += day[i]; sum += d; printf("%dn", sum); } return 0; }
