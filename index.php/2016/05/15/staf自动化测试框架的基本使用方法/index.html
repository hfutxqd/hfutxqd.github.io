    STAF自动化测试框架的基本使用方法

    @imxqd · May 15, 2016 · 1 min read

    操作步骤(以Windows为例):

    1. STAF/STAX的下载/安装和配置

      在http://staf.sourceforge.net/ 上下载并安装STAF


      在STAF的安装目录下新建services目录

      下载STAX并解压到STAF目录下的services目录


      修改STAF/bin/STAF.cfg文件,添加以下内容

    SERVICE STAX LIBRARY JSTAF EXECUTE /{STAF/Config/STAFRoot}/services/stax/STAX.jar OPTION J2=-Xmx384m

    SERVICE EVENT LIBRARY JSTAF EXECUTE /{STAF/Config/STAFRoot}/services/stax/STAFEvent.jar

    SET MAXQUEUESIZE 10000

    至此STAF/STAX配置完成

    1. 新建并执行Job

      在STAFservicesstax目录下,执行:

    Java –jar STAXMon.jar

    如下图:


    首次打开会有配置窗口:


    使用默认配置,直接点[Save]保存,弹出当前的jobs窗口:


    通过xml新建一个dir命令的job:


     

    点击[Submit New Job..],在点击Job Info Tab页的[Browse..],选择刚刚的1.xml文件

    再点击[Submit New Job],如图:




    在试试一个更简单的job,打开notepad记事本:

    2.xml
     

    
<?xml version=”1.0″ encoding=”UTF-8″ standalone=”no”?>

<!DOCTYPE stax SYSTEM “stax.dtd”>

<stax>

<defaultcall function=”main”/>

<function name=”main”>

<process>

<location>’local'</location>

<command>’notepad'</command>

</process>

</function>

</stax>

    运行结果如下:


     

    由于STAF/STAX主要功能由Java实现,在JVM中运行,所以Linux下的配置与运行方式基本与Windows无异.

    至于使用两台PC进行远程执行jobs的过程也和上面一样,只是IP地址不同而已
